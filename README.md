# testmine — 扫雷游戏

一个用 Python + tkinter 实现的经典扫雷（Minesweeper）图形界面游戏。

## 功能特性

- 🖱️ 左键单击打开格子；右键单击循环切换标记状态（🚩 旗帜 → ❓ 疑问 → 无标记）
- 🔢 支持和弦操作：在已揭开的数字格子上左键单击，若周围旗帜数等于格子数字，自动揭开剩余未标记格子
- 🌊 空白格子自动展开（Flood Fill）
- ⏱️ 游戏计时器，首次点击时启动，结束时停止
- 🏆 最佳成绩本地持久化（保存于 `~/.minesweeper_scores.json`）
- 🎮 三种预设难度 + 自定义难度
- 🛡️ 首次点击保证不踩雷，且自动筛选出无需猜测即可完全推理求解的棋盘（可解性保证）

## 环境要求

- Python 3.6+
- tkinter（Python 标准库，通常随 Python 一同安装）

## 运行方式

```bash
python minesweeper.py
```

## 难度设置

| 难度 | 棋盘大小 | 地雷数 |
|------|---------|--------|
| 初级 | 9 × 9   | 10     |
| 中级 | 16 × 16 | 40     |
| 高级 | 16 × 30 | 99     |
| 自定义 | 5–30 × 5–30 | 自定义 |

通过菜单栏 **游戏** 可切换难度或查看最佳成绩。

## 操作说明

| 操作 | 效果 |
|------|------|
| 左键单击未揭开格子 | 打开格子 |
| 右键单击未揭开格子 | 循环切换：旗帜 🚩 → 疑问 ❓ → 无标记 |
| 左键单击已揭开数字格子 | 和弦操作（周围旗帜数匹配时自动揭开剩余格子） |
| 点击顶部 🙂 按钮 | 重新开始游戏 |
